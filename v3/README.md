# V3 – Desktop-App (PySide6)

## Kurzstart
1. `start_v3.bat` ausführen.
2. Settings öffnen (API Keys eintragen).
3. Datei öffnen (`closecaption_english.txt`).
4. Bearbeiten/Auto-Translate nutzen.
5. Exportieren.

## Hinweise
* Die V3-App nutzt ein Dark-Theme und protokolliert Fehler unter `v3/logs/app.log`.
